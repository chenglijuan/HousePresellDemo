/**************************************************************************************************/
	关于权限涉及到的数据库表名如下：
	1)	权限数据核心表
		Sm_Permission_UIResource--存放本系统所有的权限资源（链接资源、按钮资源、实体菜单资源、虚拟菜单资源）
		Rel_Parent_Children_UIResource--存放（链接资源、按钮资源、实体菜单资源、虚拟菜单资源）这些权限资源的父级和子级的关联关系 
	2) 角色和权限资源的关系表
		Rel_Role_UIResource--存放角色和权限数据的关联关系
	3) 范围授权相关表
		Sm_Permission_RangeAuth--范围授权列表页面所需数据
		Sm_Permission_Range--范围授权详情页面所需数据
		Rel_Parent_RangeAuth_Range--范围授权列表页面数据和详情页面数据的关联关系
/**************************************************************************************************/
